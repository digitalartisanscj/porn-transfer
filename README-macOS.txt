PORN TRANSFER - macOS Installation
===================================

1. Open the DMG file
2. Drag the app to Applications
3. Open Terminal and run:

   sudo xattr -rd com.apple.quarantine /Applications/Porn\ Sender.app

   (or Porn\ Receiver.app for receiver)

4. Enter your password when prompted
5. Open the app normally from Applications

You only need to do step 3-4 once after installation.
